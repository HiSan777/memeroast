# MemeRoast User Guide

**MemeRoast - Roast your wallet. Go viral.**

MemeRoast is an AI Meme & Roast Agent on Arc Testnet. Connect a wallet, pick a savage personality, preview the roast for free, then send a tiny testnet USDC nanopayment to create a final meme image and caption.

## 1. Quick Start

Open:

```text
https://memeroast.xyz
```

1. Click `Connect Wallet`.
2. Choose MetaMask, Rabby, or another RainbowKit wallet.
3. Switch to Arc Testnet.
4. Chain ID: `5042002`.
5. Click `Faucet` in the header to open Circle Faucet for test USDC.

## 2. Create a Meme

1. Pick a personality: `Roast Master`, `Meme Lord`, `Crypto Degenerate`, or `Chill Girl`.
2. Enter a prompt, for example:

```text
Roast my wallet
Turn my empty portfolio into a viral meme
Make a meme about a Bitcoin crash
```

3. Optionally upload an image.
4. Click `Generate Preview`.
5. Review the English caption and meme direction.
6. Click `Pay 0.05 USDC & Generate Final`.
7. Confirm the nanopayment modal.
8. Receive the final meme image, caption, Arc Testnet transaction hash, and history entry.

## 3. Features

- `Generate`: AI wallet roast, caption, and clean meme visual.
- `My History`: Wallet-scoped local history.
- `Leaderboard`: Funny demo top memes with rank highlights.
- `Share on X`: Opens an X intent with caption-only text.
- `Copy Image`: Copies a PNG meme image when the browser allows it.
- `Download Image`: Saves the meme as PNG if image clipboard access is blocked.
- `Docs`: Public guide, roadmap, and integration notes.
- `Faucet`: Opens Circle Faucet for Arc Testnet USDC.

## 4. Important Notes

- The final generation price is displayed as `0.05 USDC`.
- Final generation sends a real Arc Testnet native USDC transfer.
- Final metadata can be anchored through the MemeRoastHistory contract on Arc Testnet.
- Meme images are generated by a mock Flux-style renderer.
- Captions are kept separate from the image so text stays readable.
- Share text is clean creator content. No automatic MemeRoast footer is added.
- Real Arc Blueprints execution, Circle USDC nanopayment, IPFS storage, and on-chain history are planned production steps.
- Never enter a private key or seed phrase. Use wallet connection only.
